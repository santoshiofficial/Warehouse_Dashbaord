Archival of GST Return Data in GST Portal
As per the GST portal's data policy, return data is accessible only for seven years. Returns older than this will be archived monthly. Returns for July 2017 were archived on August 1, 2024, August 2017 returns on September 1, 2024, and September 2017 data will be archived on October 1, 2024. We recommend downloading the data from the portal for future reference before it is archived.
